# About Drively

Drively is an open-source code that uses mobile phone as a steering wheel and pedals for gaming and simulation. 

# Requirements 
1. Install [ViGEm Bus Driver](https://github.com/nefarius/ViGEmBus/releases)
2. Restart PC

# Installation & Usage
1. Download Windows .exe and Android .apk from releases. 
2. Execute Windows .exe and install
3. The same for Android. 
4. Once installed, execute drively.exe in Windows and got to Link section. 
5. Press Init Service. 
6. Once you accepted Firewall permissions, it will show you an IP, a Port, and also a QR code too.
7. Now, go to your recently installed Android .apk
8. Go to Link section and scan QR or enter the PC previously shown data. 
9. Press connect. 
10.  Now, your device is connected and sending data. You can start using controls by pressing "Show Controls" on your Android screen. 
11.  You can also disconnect by pressing "Disconnect". 

> Dont forget to disconnect manually clicking "Close" from PC controls. Otherwise internet socket will remain open.
